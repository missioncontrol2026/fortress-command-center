Step 1 · Install the Chrome bridge extension

Required for CoStar + Reonomy pulls (uses your logged-in browser so no separate credentials or bot-blocks).

  1. ⬇ Download fortress-mc-bridge.zip
  2. Unzip to a folder you'll keep (e.g. ~/Desktop/fortress-mc-bridge/).
  3. Open Chrome → chrome://extensions
  4. Toggle Developer mode (top right)
  5. Click Load unpacked → select the unzipped folder
  6. Pin the extension. It shows a green dot when connected to the scraper backend.
  7. Make sure you're logged in to CoStar + Reonomy in the same Chrome profile.
